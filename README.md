# Excel 合并工具

一个基于浏览器的 Excel 文件合并工具，通过 5 步向导完成两个 Excel 文件的关联合并。

## 功能特点

- **文件上传** — 支持拖拽上传，接受 `.xlsx` / `.xls` / `.csv` 格式
- **多工作表** — 可选择单张或多张工作表合并，支持自定义表头行
- **关键列匹配** — 支持联动模式（所有工作表共用同一关键列）和逐表模式（各工作表独立选择）
- **列选择** — 自由勾选需要合并的列，支持搜索过滤
- **结果处理** — 三个标签页分别展示：匹配数据、未匹配数据（可勾选纳入）、冲突数据（多对多重复，支持保留全部/保留首条/移除）
- **导出** — 下载为 Excel (.xlsx) 或 CSV（含 UTF-8 BOM，兼容中文）

## 使用方式

无需安装，直接在浏览器中打开 `index.html` 即可使用。

## 技术栈

- 纯 HTML / CSS / JavaScript，单文件，无构建步骤
- [SheetJS (xlsx)](https://sheetjs.com/) v0.20.3 — Excel 读写
